# WP-project

EcoDIY Hub – A Platform for Eco-Friendly DIY Projects

The objective of this project is to create a web platform where users can explore eco-friendly DIY ideas, submit their own projects, and promote sustainable practices. The system also includes an admin panel to review and approve submitted projects.

Frontend: HTML, CSS, JavaScript
Storage: LocalStorage (Browser)
Tools: VS Code 

Project Modules:

1. Home Page
2. Guides Page (View Projects)
3. Submit Page (User Submission)
4. Admin Login & Dashboard
5. About Page

The project is divided into four modules based on dependency:

Midhun Madhu: Developed the basic structure using HTML for all pages.

Abhinav Nair: Designed the user interface using CSS including layout, cards, and responsiveness.

Aditya Nair: Implemented user-side functionalities like submitting projects, displaying approved projects, and like system using JavaScript.

Lavanya Nair: Developed admin functionalities including login system, project approval, editing, and deletion using JavaScript and LocalStorage.
